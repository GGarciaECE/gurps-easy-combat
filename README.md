# GURPS Easy Combat

## Goal: the module aims to streamline combat for the in the GURPS system by providing popups to choose maneuvers, attacks, and defenses, and more.

## Dependencies:
- socketlib

## implemented features:
- maneuver selector
- attack selector
- defense selector
